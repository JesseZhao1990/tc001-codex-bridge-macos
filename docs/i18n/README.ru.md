# TC001 Codex Bridge for macOS

[← English](../../README.md) | [Полное руководство](USAGE.ru.md)

## Обзор

TC001 Codex Bridge — нативное приложение macOS, которое показывает на Ulanzi TC001 лимиты Codex за 5 часов и 7 дней, а также рабочее состояние. Поддерживаются AWTRIX по HTTP и Bluetooth-транспорт дополнительной прошивки.

## Основные возможности

Приложение выводит две шкалы лимитов, чередующиеся значения 5H/7D и четырёхцветный индикатор. Также настраиваются страницы времени, даты, температуры, влажности и батареи. Анализ Codex выполняется локально на Mac.

## Требования

Нужны macOS 13 или новее, авторизованный Codex Desktop или CLI и TC001 с AWTRIX 3. Для Bluetooth требуется прошивка awtrix3-ble.

## Быстрый старт

1. Запустите `./run-tests.sh` и `./build.sh`.
2. Откройте `dist/TC001 Bridge.app` и разрешите Bluetooth и локальную сеть.
3. Выберите Автоматически, Wi-Fi или Bluetooth.

- Для Wi-Fi укажите IP или `awtrix.local`; для Bluetooth дождитесь AWTRIX-BLE.
- Включите автоматическое наблюдение Codex и проверьте цвета индикатора.
- Настройте пять встроенных страниц AWTRIX.

## Конфиденциальность и безопасность

Телеметрии и серверов проекта нет. Приложение читает только локальное состояние Codex и отправляет TC001 готовые пиксели и настройки страниц. Локальный API слушает 127.0.0.1 и отклоняет запросы из браузеров.
