# TC001 Codex Bridge for macOS - Полное руководство

[← README](README.ru.md) | [English usage guide](../USAGE.md)

## Установка и подключение

1. Запустите `./run-tests.sh` и `./build.sh`.
2. Откройте `dist/TC001 Bridge.app` и разрешите Bluetooth и локальную сеть.
3. Выберите Автоматически, Wi-Fi или Bluetooth.
4. Для Wi-Fi укажите IP или `awtrix.local`; для Bluetooth дождитесь AWTRIX-BLE.
5. Включите автоматическое наблюдение Codex и проверьте цвета индикатора.
6. Настройте пять встроенных страниц AWTRIX.

## Значение индикации

Левая шкала 1x8 показывает остаток за 5 часов, правая — за 7 дней. 5H отображается 7 секунд, 7D — 3 секунды. Жёлтый означает простой, зелёный работу, синий ожидание подтверждения, красный ошибку.

## Устранение неполадок

Если BLE не найден, проверьте прошивку 0.98-ble.4, разрешения и старые пары. Для Wi-Fi проверьте доступность TC001 с Mac. Если лимиты отсутствуют, проверьте вход в Codex и дождитесь обновления.

## Конфиденциальность и безопасность

Телеметрии и серверов проекта нет. Приложение читает только локальное состояние Codex и отправляет TC001 готовые пиксели и настройки страниц. Локальный API слушает 127.0.0.1 и отклоняет запросы из браузеров.
