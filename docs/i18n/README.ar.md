# TC001 Codex Bridge for macOS

[← English](../../README.md) | [دليل الاستخدام الكامل](USAGE.ar.md)

## نظرة عامة

TC001 Codex Bridge تطبيق أصلي لنظام macOS يعرض حدود Codex لمدة 5 ساعات و7 أيام وحالة العمل على Ulanzi TC001. يدعم AWTRIX عبر HTTP واتصال Bluetooth الذي توفره البرمجية الثابتة المصاحبة.

## الميزات الرئيسية

يعرض شريطي الحصة وقيم 5H و7D بالتناوب ومؤشراً بأربعة ألوان، كما يضبط صفحات الوقت والتاريخ والحرارة والرطوبة والبطارية. تتم معالجة حالة Codex محلياً على جهاز Mac.

## المتطلبات

يلزم macOS 13 أو أحدث، وCodex Desktop أو CLI مع تسجيل الدخول، وجهاز TC001 يعمل بـ AWTRIX 3. يتطلب Bluetooth برمجية awtrix3-ble المصاحبة.

## بدء سريع

1. شغّل `./run-tests.sh` ثم `./build.sh`.
2. افتح `dist/TC001 Bridge.app` وامنح أذونات Bluetooth والشبكة المحلية.
3. اختر تلقائي أو Wi-Fi أو Bluetooth.

- في Wi-Fi أدخل عنوان IP أو `awtrix.local`؛ وفي Bluetooth انتظر اتصال AWTRIX-BLE.
- فعّل المراقبة التلقائية لـ Codex واختبر ألوان المؤشر.
- اضبط صفحات AWTRIX الخمس المدمجة حسب الحاجة.

## الخصوصية والأمان

لا توجد بيانات قياس أو خادم تابع للمشروع. يقرأ التطبيق حالة Codex المحلية فقط ويرسل وحدات البكسل الجاهزة وإعدادات الصفحات إلى TC001. تستمع الواجهة المحلية على 127.0.0.1 وترفض طلبات مصدر المتصفح.
