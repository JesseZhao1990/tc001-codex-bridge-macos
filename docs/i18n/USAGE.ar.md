# TC001 Codex Bridge for macOS - دليل الاستخدام الكامل

[← README](README.ar.md) | [English usage guide](../USAGE.md)

## التثبيت والاتصال

1. شغّل `./run-tests.sh` ثم `./build.sh`.
2. افتح `dist/TC001 Bridge.app` وامنح أذونات Bluetooth والشبكة المحلية.
3. اختر تلقائي أو Wi-Fi أو Bluetooth.
4. في Wi-Fi أدخل عنوان IP أو `awtrix.local`؛ وفي Bluetooth انتظر اتصال AWTRIX-BLE.
5. فعّل المراقبة التلقائية لـ Codex واختبر ألوان المؤشر.
6. اضبط صفحات AWTRIX الخمس المدمجة حسب الحاجة.

## معنى العرض

الشريط الأيسر 1x8 يمثل رصيد 5 ساعات، والأيمن رصيد 7 أيام. تظهر 5H لمدة 7 ثوان و7D لمدة 3 ثوان. الأصفر خامل، الأخضر يعمل، الأزرق ينتظر التأكيد، والأحمر خطأ.

## استكشاف الأخطاء

إذا لم يظهر BLE فتحقق من الإصدار 0.98-ble.4 والأذونات والاقترانات القديمة. في Wi-Fi تأكد من وصول Mac إلى TC001. إذا غابت الحصة فتأكد من تسجيل الدخول إلى Codex وانتظر التحديث.

## الخصوصية والأمان

لا توجد بيانات قياس أو خادم تابع للمشروع. يقرأ التطبيق حالة Codex المحلية فقط ويرسل وحدات البكسل الجاهزة وإعدادات الصفحات إلى TC001. تستمع الواجهة المحلية على 127.0.0.1 وترفض طلبات مصدر المتصفح.
