# TC001 Codex Bridge for macOS - সম্পূর্ণ ব্যবহার নির্দেশিকা

[← README](README.bn.md) | [English usage guide](../USAGE.md)

## ইনস্টল ও সংযোগ

1. `./run-tests.sh` এবং `./build.sh` চালান।
2. `dist/TC001 Bridge.app` খুলে Bluetooth ও স্থানীয় নেটওয়ার্ক অনুমতি দিন।
3. Automatic, Wi-Fi বা Bluetooth নির্বাচন করুন।
4. Wi-Fi-তে IP বা `awtrix.local` দিন; Bluetooth-এ AWTRIX-BLE সংযোগের অপেক্ষা করুন।
5. Codex স্বয়ংক্রিয় পর্যবেক্ষণ চালু করে লাইটের রঙ পরীক্ষা করুন।
6. প্রয়োজনমতো পাঁচটি AWTRIX বিল্ট-ইন পেজ ঠিক করুন।

## ডিসপ্লের অর্থ

বাম 1x8 বার ৫ ঘণ্টার অবশিষ্ট কোটা এবং ডান বার ৭ দিনের কোটা দেখায়। 5H ৭ সেকেন্ড ও 7D ৩ সেকেন্ড দেখা যায়। হলুদ নিষ্ক্রিয়, সবুজ কাজ, নীল নিশ্চিতকরণের অপেক্ষা এবং লাল ত্রুটি।

## সমস্যা সমাধান

BLE না পেলে ফার্মওয়্যার 0.98-ble.4, অনুমতি ও পুরোনো পেয়ারিং দেখুন। Wi-Fi-তে Mac থেকে TC001 পৌঁছানো যায় কি না পরীক্ষা করুন। কোটা না এলে Codex লগইন নিশ্চিত করে রিফ্রেশের অপেক্ষা করুন।

## গোপনীয়তা ও নিরাপত্তা

কোনো টেলিমেট্রি বা প্রকল্প সার্ভার নেই। অ্যাপ শুধু স্থানীয় Codex অবস্থা পড়ে এবং TC001-এ রেন্ডার করা পিক্সেল ও পেজ সেটিং পাঠায়। স্থানীয় API 127.0.0.1-এ শোনে এবং ব্রাউজার Origin অনুরোধ প্রত্যাখ্যান করে।

## লাইসেন্স

macOS অ্যাপ MIT লাইসেন্সে প্রকাশিত। প্রকল্পটি OpenAI, Codex, Ulanzi, AWTRIX বা Blueforcer-এর সঙ্গে আনুষ্ঠানিকভাবে যুক্ত বা অনুমোদিত নয়।
