# TC001 Codex Bridge for macOS

[← English](../../README.md) | [সম্পূর্ণ ব্যবহার নির্দেশিকা](USAGE.bn.md)

## পরিচিতি

TC001 Codex Bridge একটি নেটিভ macOS অ্যাপ, যা Ulanzi TC001-এ Codex-এর ৫ ঘণ্টা ও ৭ দিনের সীমা এবং কাজের অবস্থা দেখায়। এটি AWTRIX HTTP ও সহায়ক ফার্মওয়্যারের Bluetooth পরিবহন সমর্থন করে।

## প্রধান বৈশিষ্ট্য

অ্যাপটি দুই পাশে কোটা বার, পালাক্রমে 5H/7D মান এবং চার রঙের স্ট্যাটাস লাইট দেখায়। সময়, তারিখ, তাপমাত্রা, আর্দ্রতা ও ব্যাটারি পেজও নিয়ন্ত্রণ করা যায়। Codex বিশ্লেষণ Mac-এ স্থানীয়ভাবে হয়।

## প্রয়োজনীয়তা

macOS 13 বা নতুন, লগইন করা Codex Desktop বা CLI এবং AWTRIX 3 চালানো TC001 দরকার। Bluetooth-এর জন্য awtrix3-ble ফার্মওয়্যার প্রয়োজন।

## দ্রুত শুরু

1. `./run-tests.sh` এবং `./build.sh` চালান।
2. `dist/TC001 Bridge.app` খুলে Bluetooth ও স্থানীয় নেটওয়ার্ক অনুমতি দিন।
3. Automatic, Wi-Fi বা Bluetooth নির্বাচন করুন।

- Wi-Fi-তে IP বা `awtrix.local` দিন; Bluetooth-এ AWTRIX-BLE সংযোগের অপেক্ষা করুন।
- Codex স্বয়ংক্রিয় পর্যবেক্ষণ চালু করে লাইটের রঙ পরীক্ষা করুন।
- প্রয়োজনমতো পাঁচটি AWTRIX বিল্ট-ইন পেজ ঠিক করুন।

## গোপনীয়তা ও নিরাপত্তা

কোনো টেলিমেট্রি বা প্রকল্প সার্ভার নেই। অ্যাপ শুধু স্থানীয় Codex অবস্থা পড়ে এবং TC001-এ রেন্ডার করা পিক্সেল ও পেজ সেটিং পাঠায়। স্থানীয় API 127.0.0.1-এ শোনে এবং ব্রাউজার Origin অনুরোধ প্রত্যাখ্যান করে।

## লাইসেন্স

macOS অ্যাপ MIT লাইসেন্সে প্রকাশিত। প্রকল্পটি OpenAI, Codex, Ulanzi, AWTRIX বা Blueforcer-এর সঙ্গে আনুষ্ঠানিকভাবে যুক্ত বা অনুমোদিত নয়।
