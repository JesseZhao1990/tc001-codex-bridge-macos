# TC001 Codex Bridge for macOS

[← English](../../README.md) | [पूरा उपयोग निर्देश](USAGE.hi.md)

## परिचय

TC001 Codex Bridge एक मूल macOS ऐप है जो Ulanzi TC001 पर Codex की 5 घंटे और 7 दिन की सीमा तथा काम की स्थिति दिखाता है। यह AWTRIX HTTP और सहायक फर्मवेयर के Bluetooth ट्रांसपोर्ट दोनों को समर्थन देता है।

## मुख्य सुविधाएँ

ऐप दोनों ओर सीमा बार, बारी-बारी से 5H/7D मान और चार रंग की स्थिति लाइट दिखाता है। समय, तारीख, तापमान, नमी और बैटरी पेज भी बदले जा सकते हैं। Codex का विश्लेषण Mac पर स्थानीय रूप से होता है।

## आवश्यकताएँ

macOS 13 या नया, लॉगिन किया हुआ Codex Desktop या CLI, और AWTRIX 3 वाला TC001 आवश्यक है। Bluetooth के लिए awtrix3-ble सहायक फर्मवेयर चाहिए।

## त्वरित शुरुआत

1. `./run-tests.sh` और `./build.sh` चलाएँ।
2. `dist/TC001 Bridge.app` खोलें और Bluetooth व स्थानीय नेटवर्क अनुमति दें।
3. Automatic, Wi-Fi या Bluetooth चुनें।

- Wi-Fi में IP या `awtrix.local` दें; Bluetooth में AWTRIX-BLE कनेक्शन की प्रतीक्षा करें।
- Codex की स्वचालित निगरानी चालू करके लाइट के रंग जाँचें।
- ज़रूरत के अनुसार पाँच AWTRIX पेज सेट करें।

## गोपनीयता और सुरक्षा

कोई टेलीमेट्री या परियोजना सर्वर नहीं है। ऐप केवल स्थानीय Codex स्थिति पढ़ता है और TC001 को तैयार पिक्सेल व पेज सेटिंग भेजता है। स्थानीय API 127.0.0.1 पर सुनता है और ब्राउज़र Origin अनुरोध अस्वीकार करता है।
